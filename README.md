脳内のホルモンと感情について

プラチックの感情の環は、感情が16種類に分類されている。
脳は3次元なので、3次元の値（座標・ベクトル）で表せるはず。
扁桃体で感情を司っているので、間にある海馬に注目した。
具体的には、海馬の中に言語や概念の印象が詰まっていると仮定した。

・ノルアドレナリン

不快の感情を強める

→うつ、パニック症に

・ドーパミン

・セロトニン
